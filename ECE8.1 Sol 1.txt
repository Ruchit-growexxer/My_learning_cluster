12. Pagination Formula
Page 1
size = 10
from = ( 1 - 1 ) * size = 0

Page 2
size = 10
from = ( 2 - 1 ) * size = 10

Page 3
size = 10
from = ( 3 - 1 ) * size = 20

Page N
size = 10
from = ( N - 1 ) * size
